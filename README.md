# 🤖 Scool Bot

Scool Bot est un bot Discord développé en TypeScript avec Node.js et MongoDB.

---

## 🚀 Fonctionnalités

* 🔗 Connexion à Discord via Discord.js
* 🗄️ Utilisation d’une base de données MongoDB (Atlas)
* ⚙️ Système modulaire (commandes et events)
* 📦 Facile à modifier et à étendre

---

## 📌 Ajouter le bot sur un serveur Discord

1. Cliquez sur ce lien : https://discord.com/oauth2/authorize?client_id=1500944923560509450&permissions=8&integration_type=0&scope=bot+applications.commands
2. Et profitez !
